[69. x的平方根](https://leetcode-cn.com/problems/sqrtx/)&ensp;*[easy]*

### Solution One: Binary Search
```
class Solution:
    def mySqrt(self, x: int) -> int:
        l, r, ans = 0, x, -1
        while l <= r:
            mid = (l + r) // 2
            if mid * mid <= x:
                ans = mid
                l = mid + 1
            else:
                r = mid - 1
        return ans
```

### Solution Two: sqrt(x) = exp(1/2 * ln(x))
```
class Solution:
    def mySqrt(self, x: int) -> int:
        if x == 0:
            return 0
        ans = int(math.exp(0.5 * math.log(x)))
        return ans + 1 if (ans + 1) ** 2 <= x else ans
```
